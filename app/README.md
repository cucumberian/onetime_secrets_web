## Установка
### Копирование проекта
Скопируйте проект с GitHub:
```shell
git clone git@github.com:cucumberian/onetime_secrets_web.git
```
Перейдите в директорию с проектом:
```shell
cd onetimesecret_web/onetime
```

### Установка зависимостей:
```shell
npm install
```
### Запуск дев версии
```shell
npx vite
```
